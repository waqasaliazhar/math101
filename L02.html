§1

What Is a Function?

▼
Definition — Function A function is a rule that assigns to each element a ∈ A, a single element b ∈ B.

We write  f : A → B  and say "f maps A to B."

The two crucial requirements are:
① Every element in A must be assigned to something.
② Each element in A is assigned to exactly one element in B (no splitting allowed).

The key idea is unambiguity: if you give me an input from A, I should be able to tell you exactly one output in B — no guessing, no multiple answers.

Let's see this with mapping diagrams. Click each one to see why it is or isn't a function:

✓ FUNCTION
Every element in A maps to exactly one element in B. Two elements in A can share the same output — that's fine!
✗ NOT A FUNCTION
The top element in A points to two different elements in B. A function must give exactly one output per input.
✗ NOT A FUNCTION
The bottom element in A has no arrow — it's not assigned to anything. Every element in the domain must have an output.
Notation A is called the domain of f — the set of allowed inputs.
B is called the co-domain of f — the "target" set where outputs live.
§2

Domain, Co-domain & Range

▼
Definition — Range The range of f (written Rf) is the set of all actual outputs:
Rf = { f(x) : x ∈ Df }
Key distinction The range is always a subset of the co-domain:   Rf ⊆ B.

Think of B as the building where outputs could live, and Rf as the apartments that are actually occupied.

In this course, we'll generally work with real-valued functions, meaning the co-domain is ℝ (the real numbers):

f : A ⊆ ℝ → ℝ

Natural domain convention: when the domain Df isn't explicitly given, we take it to be the largest subset of ℝ for which the formula makes sense.

Example — Natural Domains
f(x) = x² Df = ℝ  (any real number works)
f(x) = x² / (x − 1) Df = ℝ \ {1}  (can't divide by zero)
f(x) = √x Df = [0, ∞)  (can't take √ of negatives)
§3

Intervals

▼
Definition — Interval An interval is a subset of ℝ that has no gaps. It contains every real number between any two of its elements.
Non-example The integers ℤ = { …, −2, −1, 0, 1, 2, … } are not an interval — they have gaps (e.g., 1.5 is missing between 1 and 2).

Types of Intervals

There are four flavors, depending on whether the endpoints are included or excluded:

(a, b) — Open interval
(a, b) = { x ∈ ℝ : a < x < b }
[a, b] — Closed interval
[a, b] = { x ∈ ℝ : a ≤ x ≤ b }
(a, b]  or  [a, b) — Half-open (mixed) intervals
(a, b] = { x ∈ ℝ : a < x ≤ b }    [a, b) = { x ∈ ℝ : a ≤ x < b }
Infinite intervals
(−∞, a)   (−∞, a]   [b, ∞)   (b, ∞)   (−∞, ∞) = ℝ

∞ is not a number, so we always use a round bracket ( next to it — you can never "include" infinity.

Example — Unions of intervals Is ℝ \ {1} an interval? No — removing the point 1 creates a gap.

However, we can write it as a union of two intervals:
ℝ \ {1} = (−∞, 1) ∪ (1, ∞)

§4

Finding Domains — Worked Examples

▼

When a function is given by a formula, its natural domain is determined by two main rules:

Domain Danger Zones 🚫 Can't divide by zero: exclude values that make a denominator = 0.
🚫 Can't take √ of negatives: the expression under a square root must be ≥ 0.
Example 1 Find the domain of   f(x) = x / (x² − 1).

The denominator is x² − 1 = (x − 1)(x + 1). This equals zero when x = 1 or x = −1.

So we exclude both:

Df = ℝ \ {±1} = (−∞, −1) ∪ (−1, 1) ∪ (1, ∞)
Example 2 — Sign Chart Method Find the domain of   g(x) = √(x² − 5x + 6).

Step 1: We need the expression under the root to be ≥ 0:

x² − 5x + 6 ≥ 0

Step 2: Factor:   x² − 5x + 6 = (x − 2)(x − 3).

Step 3: Build a sign chart. The roots are at x = 2 and x = 3:

x ( −∞, 2 ) 2 ( 2, 3 ) 3 ( 3, ∞ )
x − 2 − 0 + + +
x − 3 − − − 0 +
(x−2)(x−3) + 0 − 0 +

Step 4: We need ≥ 0, so we take where the product is positive or zero:

Dg = (−∞, 2] ∪ [3, ∞)
§5

The Floor Function

▼
Definition — Floor Function The floor function ⌊x⌋ returns the greatest integer less than or equal to x.

In other words: round down to the nearest integer (toward −∞).
Quick Examples
⌊1⌋ = 1 ⌊−2.5⌋ = −3 ⌊1.5⌋ = 1 ⌊−1.2⌋ = −2 ⌊2.79⌋ = 2 ⌊π⌋ = 3

Key property: for any integer m, ⌊m⌋ = m.
Watch out for negatives! ⌊−2.5⌋ = −3, not −2. We always round toward −∞ (down on the number line), not toward zero.
Interactive — Explore the Floor Function
1.50
⌊1.50⌋ = 1

The graph of ⌊x⌋ is a staircase: on each interval [n, n+1), the function equals n. At each integer, the function "jumps up" by 1.

Domain: D = ℝ (every real number has a floor).
Range: R⌊x⌋ = ℤ (the output is always an integer).

Fun Application: Denomination Counting

Suppose you have Rs 793 and want to break it into bills of 500, 100, 50, etc. The floor function does this naturally!

Example — Making Change Start with 793:
• ⌊793/500⌋ = ⌊1.586⌋ = 1 → use 1 × 500 bill.   Remainder: 793 − 500 = 293.
• ⌊293/100⌋ = ⌊2.93⌋ = 2 → use 2 × 100 bills.   Remainder: 293 − 200 = 93.
• ⌊93/50⌋ = ⌊1.86⌋ = 1 → use 1 × 50 bill.   Remainder: 93 − 50 = 43.
…and so on!

Practice Problems

▼
Quiz — Find the Domain
Exercise 1 Determine whether ℝ \ {1} is an interval. If not, write it as a union of intervals.
Not an interval — removing 1 creates a gap. We can write:
ℝ \ {1} = (−∞, 1) ∪ (1, ∞)
Exercise 2 Find the domain of   h(x) = 1 / √(x − 3).
Two restrictions:
① x − 3 must be ≥ 0 (for the square root) → x ≥ 3
② x − 3 must be ≠ 0 (it's in the denominator) → x ≠ 3

Combining: x > 3, so:
Dh = (3, ∞)
Exercise 3 Evaluate:   ⌊3.7⌋,   ⌊−0.1⌋,   ⌊5⌋,   ⌊−4⌋
⌊3.7⌋ = 3   ⌊−0.1⌋ = −1   ⌊5⌋ = 5   ⌊−4⌋ = −4

Remember: ⌊−0.1⌋ = −1 (not 0!) because we round toward −∞.
Exercise 4 Find the domain of   f(x) = √(2x − 1) / (x − 3).
① Square root: 2x − 1 ≥ 0 → x ≥ 1/2
② Denominator: x − 3 ≠ 0 → x ≠ 3

Combining:
Df = [1/2, 3) ∪ (3, ∞)
