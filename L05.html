§1

Direct Substitution Property

▼

The simplest way to evaluate a limit: just plug in! This works for "nice" functions.

Direct Substitution — Polynomials If f(x) is a polynomial, then for any real number c:
limx→c f(x) = f(c)
Direct Substitution — Rational Functions If f(x) = p(x)/q(x) is a rational function and q(c) ≠ 0, then:
limx→c f(x)/g(x) = f(c)/g(c)
When does this fail? Direct substitution fails when plugging in gives an indeterminate form like 0/0. In that case, you need algebraic manipulation first (factor, conjugate, etc.) to simplify before taking the limit.
Quick examples limx→3 (x² + 2x) = 9 + 6 = 15  (polynomial — just plug in!)

limx→2 (x+1)/(x−5) = 3/(−3) = −1  (rational, denominator ≠ 0 at x=2)
§2

Limit Laws

▼

Suppose limx→c f(x) = L and limx→c g(x) = M. Then:

i. Sum / Difference
lim (f ± g) = L ± M
ii. Product
lim (f · g) = L · M
iii. Quotient
lim (f / g) = L / M
(provided M ≠ 0 and g(x) ≠ 0)
iv. Power
lim (f)n = Ln
v. Root
lim √(f) = √L  (when L ≥ 0)
Critical requirement These laws only apply when both individual limits exist. If either lim f or lim g does not exist, you cannot apply the law (though the combined limit might still exist via other methods).

Applying the Laws to Graphs

The lecture uses two graphs where limx→2 f(x) = 1 and limx→2 g(x) = 1. Meanwhile, limx→1 f(x) DNE. Let's apply the laws:

Examples from the graphs i) limx→2 (f(x) + g(x)) = 1 + 1 = 2,   and   limx→2 (f(x) − g(x)) = 1 − 1 = 0.

ii) limx→1 f(x)·g(x) = DNE, because limx→1 f(x) does not exist. We cannot apply the product law.

iii) limx→2 f(x)/g(x) = 1/1 = 1  (both limits exist and M = 1 ≠ 0).

iv) limx→1⁻ (f(x))²⁰²⁵ = 2²⁰²⁵  (using the power law with the left-hand limit).

v) limx→1⁻ √(3 + g(x)) = √(3 + 3) = √6  (root law).
§3

Algebraic Techniques for Limits

▼

When direct substitution gives 0/0, we need algebraic tricks to simplify first. Here are the main techniques:

The toolbox 1. Factor and cancel — factor numerator and denominator, cancel the common factor causing 0/0.
2. Common denominator — combine fractions into one, then simplify.
3. Conjugate multiplication — multiply by (a+b)/(a+b) to eliminate square roots via the identity (a−b)(a+b) = a²−b².

Example 1: Common Denominator Technique

Example — limh→0 [1/(3+h) − 1/3] / h Direct substitution gives 0/0. Let's combine the fractions in the numerator:
  1. Combine: 1/(3+h) − 1/3 = [3 − (3+h)] / [3(3+h)] = −h / [3(3+h)]
  2. Divide by h: [−h / 3(3+h)] / h = −1 / [3(3+h)]
  3. Now take the limit: limh→0 −1/[3(3+h)] = −1/[3·3] = −1/9

Example 2: Conjugate Multiplication

Example — limt→0 [1/(t√(1+t)) − 1/t] Direct substitution: both terms blow up. Let's combine first:
  1. Common denominator: [1/(t√(1+t))] − 1/t = [1 − √(1+t)] / [t√(1+t)]
  2. Multiply by conjugate: × [1 + √(1+t)] / [1 + √(1+t)]
  3. Numerator becomes: (1)² − (√(1+t))² = 1 − (1+t) = −t
  4. Full expression: −t / [t√(1+t)(1+√(1+t))] = −1 / [√(1+t)(1+√(1+t))]
  5. Take limit as t→0: −1 / [√1 · (1+1)] = −1/(1·2) = −1/2
Exercise — Try it yourself Evaluate: limx→4 (√x − 2) / (x − 4)
Multiply by conjugate (√x + 2)/(√x + 2):
= (x − 4) / [(x − 4)(√x + 2)] = 1/(√x + 2)
As x → 4: 1/(√4 + 2) = 1/(2+2) = 1/4.
Exercise Evaluate: limx→−1 (x² + 3x + 2) / (x + 1)
Factor numerator: x² + 3x + 2 = (x+1)(x+2).
Cancel: (x+1)(x+2)/(x+1) = x + 2.
As x → −1: −1 + 2 = 1.
§4

The Squeeze Theorem

▼

Sometimes you can't compute a limit directly (even with algebra), but you can trap the function between two simpler functions whose limits you know.

Squeeze Theorem (Sandwich Theorem) If f(x) ≤ g(x) ≤ h(x) near c, and
limx→c f(x) = limx→c h(x) = L
then   limx→c g(x) = L   as well.

The function g is "squeezed" between f and h, both of which converge to L — so g has no choice but to converge to L too.

Classic Example: x²sin(1/x)

Example — limx→0 x² sin(1/x) Problem: We can't use the product law because limx→0 sin(1/x) does not exist (it oscillates wildly, as we saw in Lecture 04).

Solution using Squeeze:
  1. We know −1 ≤ sin(1/x) ≤ 1 for all x ≠ 0.
  2. Multiply all parts by x² (which is ≥ 0):   −x² ≤ x²sin(1/x) ≤ x²
  3. Both bounds have the same limit: limx→0(−x²) = 0 = limx→0(x²)
  4. By the Squeeze Theorem: limx→0 x²sin(1/x) = 0. ✓

Even though sin(1/x) oscillates forever, the x² factor "crushes" the oscillation to zero. The envelope shrinks faster than the oscillation can escape.

Another Example: Squeezing with Polynomials

Example — Squeeze with polynomial bounds Suppose 4x − 9 ≤ f(x) ≤ x² − 4x + 7 for x ≥ 0. Find limx→4 f(x).
Evaluate both bounds at x = 4:
Lower: limx→4(4x − 9) = 16 − 9 = 7
Upper: limx→4(x² − 4x + 7) = 16 − 16 + 7 = 7

Both bounds converge to 7. By the Squeeze Theorem:
limx→4 f(x) = 7

Practice Problems

▼
Quiz — Evaluate the Limit
Exercise 1 Evaluate limh→0 [(2+h)³ − 8] / h.
Expand: (2+h)³ = 8 + 12h + 6h² + h³.
So (2+h)³ − 8 = 12h + 6h² + h³ = h(12 + 6h + h²).
Divide by h: 12 + 6h + h².
As h → 0: 12.
Exercise 2 Use the Squeeze Theorem to evaluate limx→0 x⁴cos(2/x).
−1 ≤ cos(2/x) ≤ 1 → −x⁴ ≤ x⁴cos(2/x) ≤ x⁴.
Both bounds → 0 as x → 0. By Squeeze Theorem: 0.
Exercise 3 Evaluate limx→9 (√x − 3) / (x − 9).
Note x − 9 = (√x − 3)(√x + 3). Cancel:
= 1/(√x + 3). As x → 9: 1/(3+3) = 1/6.
Exercise 4 If 3x ≤ f(x) ≤ x² + 2 for all x near 3, find limx→3 f(x).
Lower: limx→3(3x) = 9.   Upper: limx→3(x²+2) = 11.
These are not equal, so the Squeeze Theorem does not apply here. We can only conclude 9 ≤ lim f(x) ≤ 11 (if the limit exists), but we can't determine the exact value without more information about f.
