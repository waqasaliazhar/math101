§1

Why Limits? — Motivating Examples

▼

Before we define limits formally, let's see why we need them. Several classical problems all hinge on the idea of doing something "forever" or "infinitely close."

Archimedes and the Circle

Take a circle, cut it into pizza slices, and rearrange them alternately. As the number of slices grows, the shape looks more and more like a rectangle with height r and width C/2 = πr. Area = r · πr = πr². But what does "ultimately" mean here?

The Rope Problem — Adding Infinitely Many Numbers

Thought Experiment Take a rope of length L. Cut it in half — keep one piece (L/2). Cut the remaining piece in half — keep one (L/4). Keep going forever.

The total length of all kept pieces is:
L/2 + L/4 + L/8 + ⋯ = L
We're adding infinitely many positive numbers and getting a finite sum! And we can get as close to L as we want by taking enough terms.

Zeno's Paradox

Zeno's Paradox (~450 BC) A man stands at distance d from a wall. Each step, he travels half the remaining distance. After step 1 he's at d/2 from the wall, after step 2 at d/4, and so on.

Zeno argued: he never reaches the wall, because there's always some distance left (d/2n > 0 for any finite n).

But physics says he does reach it! The resolution: the infinite sum of times it takes is finite. Limits make this precise.

The Tangent Line Problem

Given a distance-time curve s(t), how do we find the instantaneous velocity at time t? Newton's idea: take the secant line through (t, s(t)) and (t+h, s(t+h)), with slope:

slope = [s(t+h) − s(t)] / h

Then let h → 0. The secant line "becomes" the tangent line. But what does h → 0 actually mean? That's exactly what limits formalize.

§2

The Informal Definition of a Limit

▼
Definition — Limit (Informal) Let f : A → ℝ be a function and c a real number (f may or may not be defined at c). We say
limx→c f(x) = L
if f(x) gets arbitrarily close to L whenever x is sufficiently close to c (but x ≠ c).
Critical point The limit is about what happens near c, not at c. The function doesn't even need to be defined at c for the limit to exist!
Example — Limit via a table Consider f(x) = (x² − 1)/(x − 1) at c = 1. Note that f(1) is undefined (0/0).

Let's see what happens as x approaches 1 from both sides:

For x ≠ 1:   (x²−1)/(x−1) = (x−1)(x+1)/(x−1) = x+1.
As x → 1, x+1 → 2. ✓
§3

Tables Can Be Misleading!

▼
Cautionary tale Consider f(x) = sin(π/x) and the question: what is limx→0 sin(π/x)?

If we naively build a table using x = 1, 1/2, 1/3, 1/4, …:

xsin(π/x)
10
1/20
1/30
1/40
1/1000
1/100000

You might guess the limit is 0. But that's wrong!

The table only tested inputs of the form 1/n, where sin(nπ) = 0 always. There are plenty of other inputs near 0 where sin(π/x) is not zero — it oscillates wildly between −1 and 1.

The function sin(π/x) oscillates infinitely often near 0, so limx→0 sin(π/x) does not exist.

Lesson Tables can suggest a limit by showing a pattern, but they can never prove it. You might accidentally pick special inputs that all give the same output. A rigorous proof requires the ε-δ definition.
§4

The ε-δ Definition — A Game

▼

We're going to make limits precise using a clever two-player game.

The ε-δ Game Player 1 (the challenger) picks any ε > 0 and draws a horizontal band (L−ε, L+ε) around L.

Player 2 (the defender) must find a δ > 0 such that for every x in (c−δ, c+δ) with x ≠ c, we have f(x) ∈ (L−ε, L+ε).

If the defender can always win (for every ε, no matter how small), then limx→c f(x) = L.

If the challenger finds even one ε where the defender can't respond, then the limit either doesn't equal L, or doesn't exist.
Formal Definition We write limx→c f(x) = L if:
For every ε > 0, there exists a δ > 0 such that:
if 0 < |x − c| < δ, then |f(x) − L| < ε

Let's unpack this with pictures. The condition |x − c| < δ means x ∈ (c−δ, c+δ). The condition |f(x) − L| < ε means f(x) ∈ (L−ε, L+ε).

Interactive — Play the ε-δ Game with f(x) = 2x, c = 1, L = 2

You're the challenger — pick ε. The defender responds with δ = ε/2.

1.00
Worked example — Why δ = ε/2 works for f(x) = 2x Claim: limx→1 2x = 2. We show that δ = ε/2 works for any ε > 0.
Given ε > 0, let δ = ε/2. Suppose 0 < |x − 1| < δ.

Then:
|f(x) − 2| = |2x − 2| = 2|x − 1| < 2δ = 2 · (ε/2) = ε.   ✓

So for any ε the challenger picks, the defender can always respond with δ = ε/2, and the outputs stay within the ε-band. The limit is 2.
Exercise — Find δ For f(x) = 3x + 1, c = 2, L = 7. Find δ in terms of ε.
|f(x) − 7| = |3x+1 − 7| = |3x − 6| = 3|x − 2| < 3δ.
We need 3δ < ε, so δ = ε/3 works.
§5

One-Sided Limits

▼

On the real line, you can approach a point c from two directions: from the left (x < c) or from the right (x > c).

Definitions — One-Sided Limits Left-hand limit:   limx→c⁻ f(x)  — approach c from below (x < c).

Right-hand limit:   limx→c⁺ f(x)  — approach c from above (x > c).
The Two-Sided Limit Exists If and Only If
limx→c f(x) exists  ⟺  limx→c⁻ f(x) = limx→c⁺ f(x)
Both one-sided limits must exist and be equal.
Example — A jump discontinuity Consider the piecewise function shown in the lecture: approaching x = 2, the function comes from the left toward 1 (with an open circle at (2,1)), but from the right it jumps to 3 (with a filled dot at (2,3)).

limx→2⁻ f(x) = 1   and   limx→2⁺ f(x) = 3.

Since 1 ≠ 3, the two-sided limit limx→2 f(x) does not exist.

Example — |x|/x

Example — limx→0 |x|/x Since |x| = x for x > 0 and |x| = −x for x < 0:
|x|/x = 1  for x > 0,   |x|/x = −1  for x < 0
So: limx→0⁺ |x|/x = 1,   limx→0⁻ |x|/x = −1.

Since 1 ≠ −1, the limit limx→0 |x|/x DNE.

§6

Practice Problems

▼
Quiz — Evaluate the Limit
Exercise 1 Using a table or algebra, guess limx→3 (x² − 9)/(x − 3).
Factor: (x²−9)/(x−3) = (x−3)(x+3)/(x−3) = x+3 for x ≠ 3.
As x → 3: x+3 → 6.
Exercise 2 For f(x) = 5x − 3, prove limx→2 f(x) = 7 using ε-δ.
|f(x) − 7| = |5x−3−7| = |5x−10| = 5|x−2| < 5δ.
Need 5δ < ε → choose δ = ε/5.
Check: if 0 < |x−2| < ε/5, then |f(x)−7| = 5|x−2| < 5·(ε/5) = ε. ✓
Exercise 3 Find the left and right limits of f(x) = ⌊x⌋ (floor function) at x = 3. Does limx→3 ⌊x⌋ exist?
From the right: For x slightly above 3 (like 3.01), ⌊x⌋ = 3. So limx→3⁺ ⌊x⌋ = 3.
From the left: For x slightly below 3 (like 2.99), ⌊x⌋ = 2. So limx→3⁻ ⌊x⌋ = 2.

Since 3 ≠ 2, the two-sided limit does not exist.
Exercise 4 Explain intuitively why limx→0 sin(π/x) does not exist, even though sin(π/x) is bounded between −1 and 1.
As x → 0, the argument π/x → ±∞, so sin(π/x) cycles through all values in [−1, 1] infinitely often. No matter how small a neighborhood of 0 you take, the function takes values near both 1 and −1, so it can't settle near any single value L.
