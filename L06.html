§1

Limits at Infinity & Comparing Growth

▼
Motivation — Newton's Law of Gravitation F = G·m₁·m₂ / r². As r → 0⁺, F → ∞ (planets colliding = infinite force). As r → ∞, F → 0 (far apart = no force). Limits at infinity and at zero capture the extremes!

When we write limx→∞ f(x), we're asking: what value (if any) does f(x) approach as x grows without bound?

Key Fact For any positive power n:   limx→∞ 1/xn = 0.

This is the workhorse for evaluating limits of rational functions at infinity: divide numerator and denominator by the highest power of x.

Comparing Growth Rates

When we take limx→∞ f(x)/g(x), the result tells us which function "wins" the growth race:

Example — Same degree limx→∞ (x²+1)/(5x²+x). Divide top and bottom by x²:
= limx→∞ (1 + 1/x²) / (5 + 1/x) = (1+0)/(5+0) = 1/5
Example — Denominator wins limx→∞ x²/(x⁴+7). Divide by x⁴:
= limx→∞ (1/x²)/(1+7/x⁴) = 0/(1+0) = 0

Teaser — Who wins: x2025²⁰²⁵ or eˣ? eˣ always wins against any polynomial, no matter the degree! Exponentials grow faster than any power of x. This remarkable fact will be made precise with L'Hôpital's rule later in the course.
§2

Vertical & Horizontal Asymptotes

▼
Vertical Asymptote The line x = c is a vertical asymptote of y = f(x) if one (or both) of the one-sided limits at c is +∞ or −∞.

The function "blows up" near x = c.
Horizontal Asymptote The line y = d is a horizontal asymptote of y = f(x) if:
limx→+∞ f(x) = d   or   limx→−∞ f(x) = d
The function "levels off" to d as x goes to ±∞.
How many can you have? A function can have infinitely many vertical asymptotes (think: tan(x) has one at every odd multiple of π/2). But it can have at most 2 horizontal asymptotes — one for x → +∞ and one for x → −∞.

§3

Finding Asymptotes — Worked Examples

▼

Finding Vertical Asymptotes

Example — Holes vs. Vertical Asymptotes Find the vertical asymptotes of f(x) = (x²−5x+6)/(x²−4x+3).
  1. Factor: f(x) = (x−2)(x−3) / [(x−1)(x−3)]
  2. Cancel common factor: for x ≠ 3, f(x) = (x−2)/(x−1)
  3. At x = 3: limx→3 f(x) = lim (x−2)/(x−1) = 1/2. Finite limit → x = 3 is a hole (removable), NOT a VA.
  4. At x = 1: limx→1⁻ (x−2)/(x−1) = (−1)/(0⁻) = +∞. limx→1⁺ = (−1)/(0⁺) = −∞. Both blow up → x = 1 is a VA. ✓

Finding Horizontal Asymptotes

Example — Different HAs in each direction Find the horizontal asymptotes of y = √(x²+1)/(x+1).

The key is that √(x²) = |x|, which equals x when x > 0 and −x when x < 0.

  1. Factor x² out of the root: √(x²+1) = √(x²(1+1/x²)) = |x|·√(1+1/x²)
  2. As x → +∞: |x| = x, so y = x·√(1+1/x²) / [x(1+1/x)] = √(1+1/x²)/(1+1/x) → √1/1 = 1. HA: y = 1.
  3. As x → −∞: |x| = −x, so y = (−x)·√(1+1/x²) / [x(1+1/x)] = −√(1+1/x²)/(1+1/x) → −1. HA: y = −1.

This function has two different horizontal asymptotes — one in each direction!

§4

Continuity

▼
Definition — Continuity at a Point A function f is continuous at c if all three conditions hold:

① f(c) is defined.
② limx→c f(x) exists.
③ limx→c f(x) = f(c).

Equivalently:   limx→c⁻ f(x) = limx→c⁺ f(x) = f(c).

Intuitively, f is continuous at c if you can draw the graph through the point (c, f(c)) without lifting your pen.

What's automatically continuous? Polynomials, rational functions (on their domain), root functions, trig functions, exponentials, and logarithms are all continuous wherever they're defined. Most "formula-based" functions are continuous — discontinuities typically arise at boundaries of piecewise definitions or where the domain has gaps.
§5

Types of Discontinuities

▼

Removable (Hole)

Limit exists, but f(c) is either undefined or ≠ limit. "Fixable" by redefining f(c).

Jump

Left and right limits both exist, but they're different. The function "jumps."

Infinite

At least one side goes to ±∞. There's a vertical asymptote at c.

Example — Removable discontinuity f(x) = (x²−1)/(x−1) at c = 1. Since f(1) is undefined (0/0), f is discontinuous at 1.

But limx→1 f(x) = lim (x+1) = 2 exists. So this is a removable discontinuity — define F(x) = f(x) for x≠1 and F(1) = 2, and F is continuous.
Example — Jump discontinuity f(x) = { x²−1 if x ≥ 0;   2x+1 if x < 0 } at c = 0.

limx→0⁻ (2x+1) = 1,   limx→0⁺ (x²−1) = −1.
Since 1 ≠ −1, this is a jump discontinuity.
Example — Finding "a" for continuity Find a so that f(x) = { 2x−1 if x ≥ 1;   ax+2025 if x < 1 } is continuous at c = 1.
For continuity, we need: limx→1⁻ = limx→1⁺ = f(1).
Right: limx→1⁺ (2x−1) = 1 = f(1).
Left: limx→1⁻ (ax+2025) = a + 2025.
Set equal: a + 2025 = 1 → a = −2024.

Extra: Homeomorphisms

Homeomorphism — Coffee cup ↔ Donut A continuous bijective map f: A → B such that f⁻¹ is also continuous is called a homeomorphism. It means you can continuously deform A into B — stretching and bending allowed, but no cutting or gluing.

The classic example: a coffee mug is homeomorphic to a donut (torus) — both have exactly one "hole" (the handle / the donut hole). A topologist can't tell them apart!
